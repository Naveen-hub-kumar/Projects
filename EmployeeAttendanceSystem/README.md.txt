# 🏢 Employee Attendance Management System

A web-based Employee Attendance Management System developed using Java, SQL, HTML, CSS, and JavaScript to efficiently track and manage employee attendance.

---

## 🚀 Features
- 🔐 User Login & Authentication  
- 📝 Mark Attendance (Check-in / Check-out)  
- 📊 View Attendance Records  
- 👨‍💼 Admin Dashboard  
- 🔍 Search Employee Records  
- 💾 Database Integration (MySQL)  

---

## 🛠️ Technologies Used
- ☕ Backend: Java (JDBC)  
- 🌐 Frontend: HTML, CSS, JavaScript  
- 🗄️ Database: MySQL  

---

## 💡 Description
This project is designed to simplify employee attendance tracking by providing an easy-to-use interface for employees and administrators. It includes features like login authentication, attendance marking, and record management. The backend handles data processing and database operations, while the frontend ensures a user-friendly experience.

---

## ▶️ How to Run
1. 📥 Clone or download the repository  
2. 🗄️ Set up MySQL database using provided SQL files  
3. ⚙️ Configure database connection in Java code  
4. ▶️ Run the Java backend server  
5. 🌐 Open `index.html` in your browser
