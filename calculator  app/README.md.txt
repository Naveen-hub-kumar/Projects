#  Calculator App

A simple and responsive calculator application built using HTML, CSS, and JavaScript to perform basic arithmetic operations.

##  Features
- + Addition
- - Subtraction
- * Multiplication
- / Division
-  Clear/reset functionality
-  Responsive design

##  Technologies Used
- HTML
- CSS
- JavaScript

##  Description
This project demonstrates core front-end development concepts such as DOM manipulation, event handling, and building interactive user interfaces. The calculator performs real-time calculations based on user input.

##  How to Run
1. Clone or download the repository
2. Open `index.html` in your browser
3. Start using the calculator

- index.html
- style.css
- script.js
- README.md
