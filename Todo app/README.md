# 📝 To-Do List Web App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript to help users manage daily tasks efficiently.

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 🔄 Real-time updates using DOM manipulation
* 📱 Responsive design for all devices

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 💡 Description

This project demonstrates basic front-end development concepts such as DOM manipulation, event handling, and dynamic UI updates. It provides a clean and user-friendly interface for managing tasks effectively.

## 📂 How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding your tasks

## 🎯 Learning Outcomes

* Improved JavaScript logic building
* Hands-on experience with DOM manipulation
* Better understanding of responsive UI design
