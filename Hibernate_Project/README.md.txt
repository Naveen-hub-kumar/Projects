# 🗄️ Hibernate CRUD Application

A simple Java-based Hibernate project that demonstrates CRUD (Create, Read, Update, Delete) operations using Hibernate ORM and MySQL database.

---

## 🚀 Features
- ➕ Insert (Save) Data  
- 🔍 Fetch (Retrieve) Data  
- ✏️ Update Records  
- ❌ Delete Records  
- 🔗 Hibernate ORM Integration  
- 💾 MySQL Database Connectivity  

---

## 🛠️ Technologies Used
- ☕ Java  
- 🗄️ Hibernate (ORM Framework)  
- 🐬 MySQL  
- ⚙️ JDBC  
- 📦 Maven / JAR Dependencies  

---

## 💡 Description
This project demonstrates how to use Hibernate ORM framework to perform database operations in a simplified way without writing complex SQL queries. It maps Java objects to database tables using annotations or XML configuration.

---

## ▶️ How to Run
1. 📥 Clone or download the repository  
2. 🗄️ Create database in MySQL  
3. ⚙️ Configure `hibernate.cfg.xml` (DB URL, username, password)  
4. ▶️ Run `App.java` (main class)  
5. ✅ Perform CRUD operations
